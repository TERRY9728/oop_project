### Getting Start

1. Clone the repository to local device or directly download all the files.

2. Run the "main.py" file directly under task1 directory.

3. You will get the loop-back address:
    ```
    >> python3 main.py
    http://127.0.0.1:8000
    ```

4. Open it in your browser.
